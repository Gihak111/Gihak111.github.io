대화 감지 기능

AI가 실시간으로 대화 내용을 분석하여 로맨스 스캠 패턴을 감지하고 경고를 제공합니다.

🛡️

백그라운드 관리

연락처 접근이나 의심스러운 URL 통신을 감지하여 개인정보 유출을 차단합니다.

🤖

AI 봇 대화 분석

대화에 참여하여 문맥을 분석하고 스캠 위험도를 실시간으로 평가합니다.

🔒

개인정보 보호

강력한 암호화와 보안 프로토콜로 사용자의 개인정보를 안전하게 보호합니다.

⚡

실시간 모니터링

24시간 실시간으로 위험 요소를 모니터링하여 즉각적인 보호 서비스를 제공합니다.

기술 아키텍처

React Native, Express Server, Flask를 활용한 안정적인 3-tier 아키텍처

메시지 모니터링, 네트워크 패킷 분석, 카메라 보안을 통한 종합적 보호

Frontend

React Native를 활용한 크로스플랫폼 모바일 앱으로 사용자 친화적인 인터페이스 제공

Backend

Express Server와 Flask를 통한 안정적인 API 서비스 및 AI 모델 운영

AI/ML

BERT, ViT 모델을 활용한 텍스트 분석 및 이미지 인식 기술
